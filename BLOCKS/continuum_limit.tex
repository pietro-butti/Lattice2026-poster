\begin{block}{Continuum limit}
    We compute the topological susceptibility $\chi$ as a function of the lattice
    spacing $a^2$, both in units of the flow scale $t_0$. This dimensionless ratio
    is sensitive to both classical and quantum cutoff effects.
    \begin{figure}
      \includegraphics[scale=0.6]{PLOTS/L64_CL.pdf}
      \includegraphics[scale=0.7]{PLOTS/cl_behaviour_t0.pdf}
    \end{figure}
    \begin{description}
        \item[\textbf{Left:}]
        {\color{coolgreen}
            The ``\emph{big volume}'' ensembles expose the $\order{a^2}$ cutoff effects
            of the action discretisation. Since the charge uses the clover definition,
            a residual ``classical'' effect contributes at the same order. With the
            statistics available, both are subleading compared to the statistical error.
            (Point opacity is proportional to the number of independent configurations
            in the ensemble.)
        }

        \item[\textbf{Right:}]
        {\color{coolblue}
            The ``\emph{fully improved}'' ensembles isolate the ``classical'' $\order{a^2}$
            effects. From a single, higher-statistics set of Lüscher--Weisz configurations
            we compare four extrapolations, spanning all combinations of Wilson/Zeuthen
            flow and clover/improved charge.
        }
    \end{description}
\end{block}
